Wind4Strava

Control wind overlay for route planning

Use the map buttons near Heatmaps and Segments: Wind toggle + Refresh.

Default is OFF to avoid unnecessary API calls. Configure first, then enable when needed.

Wind overlay

How to use

  1. Open Strava route builder (`/maps/create` or `/routes/new`).
  2. Toggle wind ON from popup or from the map wind button near Heatmaps and Segments.
  3. Move the map freely, then press the map Refresh button to load wind for the new visible area.
  4. Adjust forecast time and arrow density in this popup.

Behavior and limitations

  • Map movement redraws arrows from existing data and does not trigger automatic API calls.
  • Use the map Refresh button for explicit area updates; fresh cache is reused when available.
  • Density changes redraw arrows from cached data and do not add extra API calls for the same map area.
  • If Open-Meteo rate-limits, Wind4Strava retries and may temporarily use recent cached wind.
  • When wind is OFF, no wind API calls are made.

Troubleshooting

  • If no overlay appears, keep the Strava route tab active and reload the tab once.
  • If status says waiting for viewport, zoom or move the map a bit.
  • If rate-limited, wait briefly and try again.
